🔐 Политика конфиденциальности — Vast VPN

Общие положения 1.1. Настоящая Политика регулирует обработку данных при использовании Vast VPN (далее — «Сервис»). 1.2. Используя Сервис, Пользователь соглашается с Политикой. Сбор данных 2.1. Сервис может собирать: — идентификаторы аккаунта (например, Telegram ID, email); — технические данные (IP-адрес, устройство, браузер); — информацию о взаимодействии с Сервисом (подписки, платежи, обращения). 2.2. Сервис не требует избыточных персональных данных. Использование данных 3.1. Данные используются для: — предоставления доступа; — поддержки; — безопасности; — улучшения Сервиса. Передача третьим лицам 4.1. Данные не продаются. 4.2. Передача возможна: — платёжным системам; — техническим подрядчикам; — по требованию закона; — с согласия Пользователя. Cookies 5.1. Сервис может использовать cookie для работы сайта и аналитики. 5.2. Пользователь может отключить их в браузере. Хранение и защита 6.1. Данные хранятся в течение срока использования Сервиса и разумного периода после. 6.2. Применяются меры защиты, но абсолютная безопасность не гарантируется. Права пользователя 7.1. Пользователь может: — запросить данные; — изменить или удалить их; — отозвать согласие. Изменения 8.1. Политика может обновляться. 8.2. Продолжение использования означает согласие.
